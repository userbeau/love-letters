Commands:


  • "git init" - Initialize a local Git repository
  • "git log" - Show commit logs
  • "git log --summary" - Show detailed commit logs
  • "git status" - Show the working tree status
  • "git add filneame" - Add file to staging area
  • "git add ." - Add all files with changes to staging area
  • "git commit -m "message"" - Commit files in staging area
  • "git branch newbranchname" - Make a new branch
  • "git branch" - List all branches
  • "git branch -D" - Delete branches
  • "git checkout " - Switch branches or restore working tree files
  • "git merge" - Join two or more development histories together
