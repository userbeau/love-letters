Commands:

  • "pwd" - present working directory
  • "cd" - change directory
  • "cd ~" - go to home directory
  • "cd .." - go to parent directory
  • "ls" - print directory contents
  • "ls -la" - print directory contents with more info
  • "touch filename" - make a new file
  • "touch filename" - make a new file
  • "touch filename" - make a new file
  • "nano" - open nano text editor
  • "nano filename" - open file with nano
  • "mkdir directoryname" - make a directory
  • "rm filename" - remove a file
  • "rmdir directoryname" - remove a directory
  • "rm -R" - remove a dircetory and everything inside it
